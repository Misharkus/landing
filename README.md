Landing Page
Responsive landing page inspired by Bang &
Olufsen design.
Built as a portfolio project using modern
frontend tools.

## 🚀 Demo

👉 [Live Preview](https://Misharkus.github.io/landing/)

## 🛠️ Tech Stack

- HTML5
- SCSS (Sass)
- JavaScript (ES6)
- Parcel bundler
- GitHub Pages (deployment)

---

🎯 Features

- Fully responsive layout
- Clean and modern UI design
- Mobile menu
- Smooth section structure
- Optimized assets via Parcel

---

## 🚀 How to run locally

npm install
npm start

## 🏗 Build project

npm run build

## 🚀 Deploy

npm run deploy
